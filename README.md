# Rasanusacathering
Rasa Nusa Catering didirikan pada tahun 2020 dengan semangat untuk menghadirkan cita rasa khas Nusantara yang autentik dan berkualitas ke tengah masyarakat. Berawal dari usaha rumahan sederhana, kami mulai melayani pesanan dalam skala kecil untuk acara keluarga dan lingkungan sekitar.
